## What to Look For

### 1. Console Tab (Most Important!)
- **Red errors** = Something broke
- **Yellow warnings** = Potential issues
- **Blue info** = Usually fine

### 2. Network Tab
- Check if files are loading (green = good, red = failed)
- Look for CSS/JS files returning 404 or errors

### 3. Elements Tab
- Verify `<div id="root">` exists
- Check if React components are rendering

## Common Issues to Report

### White Screen:
- Console errors?
- Network failures?
- Missing root element?

### Black Screen:
- CSS not loading?
- JavaScript errors?
- Firebase connection issues?
