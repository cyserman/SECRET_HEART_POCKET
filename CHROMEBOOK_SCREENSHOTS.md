## Quick Screenshot Methods

### Method 1: Keyboard Shortcut (Fastest)
- **Full screen**: `Ctrl + Show windows` (or `Ctrl + F5`)
- **Partial screen**: `Ctrl + Shift + Show windows` (then drag to select area)

### Method 2: Screenshot Tool
1. Click the time in bottom right
2. Click the screenshot icon (camera)
3. Choose: Full screen, Partial, or Window

### Method 3: Chrome Extension
- Install "Awesome Screenshot" or "Nimbus Screenshot" from Chrome Web Store
